Open with Biofile Finder...

Biofile Finder will load Key-Value pairs from Images in OMERO. For large Projects, this can take time to export, but for a Project with fewer than 400-500 images, it can be done "on the fly".

Load on the fly

BFF will load the Key-Value pairs from all the images "on the fly". If you refresh the page, BFF will reload the Key-Value pairs from the server and show any updates.
